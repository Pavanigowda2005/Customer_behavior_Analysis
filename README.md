# 📊 Customer Behavior Analysis

## 📌 Overview
This project focuses on analyzing customer behavior using data analytics techniques to uncover purchasing patterns, customer trends, and actionable business insights. The workflow includes data loading, cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and report generation.

The goal is to transform raw customer data into meaningful insights that support data-driven decision-making.

---

## 📂 Dataset
The dataset contains customer-related information such as:
- Customer demographics  
- Purchase history  
- Transaction details  
- Product categories  
- Behavioral patterns  

Data preprocessing was performed to handle missing values, duplicates, and inconsistencies.
